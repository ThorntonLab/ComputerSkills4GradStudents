Installing new software¶

We have created a new venv and know a bit about what is in it. Now, let’s install some Python packages!

Using the command line¶

To install the Python seaborn plotting package:

pip install seaborn

To uninstall it:

pip uninstall seaborn

requirements.txt files¶

Create a file called requirements.txt with the following contents:

pandas
scipy

Then:

pip install -r requirements.txt

The requirements.txt file for this book is:

jupyter-book
matplotlib
numpy

What we learned¶

  • pip is a command to install Python packages.

  • We invoke pip using the command line.

  • We may keep a list if packages to install in a file. By tradition, that file name is called requirements.txt. Other names are allowed. Multiple “requirements” files for different purposes are allowed.

Read more about pip. The link contains a detailed section on requirements files. Typically, requirements.txt lists exactly what you need to get your own code running. For example, if your code makes a bunch of plots using seaborn, then simply write seaborn in the file. (After reading one of the previous links, feel free to specify an exact or minimal seaborn version.)
